Kaleidoscope is an interactive visual toy which uses touch on your handheld device to control the output on the screen.

Add shapes. Select a color scheme. Abstract it. Have fun!

Here's a cheat sheet for the top level of controls.

Mirrors
Turns the kaleidoscope mirrors on/off. Off by default.
Pixelate
Adds a semi-transparent, pixelated layer of the kaleidoscope.
Bleed
Most awesome mode. Let the colors bleed! Shapes will leave fun trails! Off by default.
Color Themes
Change the color theme. Theme 0 by default. Overlap themes when Bleed is enabled!
Shapes
Pauses the kaleidoscope and opens an edit drawer of options, including Color Themes. Add shapes! Comes complete with a mystery "?" shape!
Minimize
Minimizes the menu. Get a better look at the kaleidoscope!
